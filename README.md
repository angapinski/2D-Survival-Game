# 2D-Survival-Game
Course project for my 2D Graphics course. This is a 2D survival game which  includes procedural map generation.

## Block Types ##
### Generic Tiles ###
- Dirt
- Dirt (w/ grass on surface)
- Stone
- Sand
- Sandstone
- Clay

### Liquid Tiles ###
- Ocean
- Water
- Lava

### Special Tiles ###
- Neutronium (Bottom of map)
